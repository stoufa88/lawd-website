The introduction

Watching a movie instantly is fast and fun

Lawd is a centralized hub for torrents around the web. It uses the developer api from http://themoviedb.org to list all Movies and TV shows. Since we don't want to wait for download to complete and we want to share what I'm watching with others, so we attach the magnet and different subtitles to a show in the catalog. The catalog becomes a complete streaming solution.

This application don't have a built in VPN. Use it at your own risk, since we know torrents can contain pirated movies and TV shows.
