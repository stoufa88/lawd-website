The introduction

Watching instantly is fast and fun

Lawd is a centralized hub for torrents around the web. It uses the developer api from http://themoviedb.org to list all Movies and TV shows. You just browse and pick the show you want to watch. If you can't find your show, ask for it or browse the web to find the source and add it in here. With this kind of collaboration, Lawd become the best solution for community shared torrents.

This application don't have a built in VPN. Since we know torrents can contain pirated movies and TV shows, use it at your own risk.
